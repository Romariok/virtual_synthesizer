VIRTUAL SYNTHESIZER
===================
# Проект по Языкам Программирования (2023-24) 2 курс 
Виртуальный синтезатор с аддитивным синтезом

# Управление
* Нижняя половина - первая октава
* Верхняя половина - вторая октава
* Стрелочки - менять звук нот
